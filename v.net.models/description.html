key* [Name of integer key field in input points' attribute table]
This option must be set to the name of an attribute of type integer that serves to uniquely identify each point in the input map. The field must contain a unique value for each point (node), and it must not be the primary key field that GRASS uses by default (normally: "cat"). To guarantee accurate operation, this option should always be set to the same value as it was when running v.net.models.
